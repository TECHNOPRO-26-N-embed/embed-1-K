## 4. 一番苦労したこと、どう乗り越えたか（30秒）

<!-- ここが発表の山場です。1つだけに絞って、ストーリーで書いてください -->

**何が起きたか：**
- 設計書をもとに回路を組み立て、プログラムを書き込んだが、シリアルモニターに現在のシステム状態が連続で表示され、他の操作ができないトラブルが発生した


**どう対処したか：**
- まずAIに相談して、ソースコードの説明書きを入れてもらい、表示のタイミングを調整する方法を教えてもらい、必要なときだけ表示するようにコードを修正しました。
- その結果、シリアルモニターに必要なときだけ必要な情報が表示されるようになり、他の操作もできるようになった

**そこから何がわかったか：**
- AIが生成したコードをそのまま使うと、正しく動かないことがある。自分でコードを理解して、必要に応じて修正することが重要。
